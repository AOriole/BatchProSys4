Batch Processing System for Banking Transactions



Qucik Start
  1. Click load project
  2. Click Upload Accounts tab & select "accounts.json" file
  3. Click Upload Transactions tab & select "transactions.txt" file
  4. Click Process Transaction tab to process the transactions
  5. Click Analyse Transaction tab to visualize the transactions
